# Paper Analysis

## 适用场景

用户提供 PDF 路径、论文文本、摘要或关键章节，希望得到单篇论文的贡献、方法、实验和不足分析。

## 输入格式

- PDF 路径。
- Markdown/TXT/LaTeX 文本。
- 摘要、方法段或实验段。

## 输出格式

- 基本信息。
- 一句话总结。
- 研究问题。
- 核心贡献。
- 方法详解。
- 实验分析。
- 可复现性评估。
- 优点、缺点和审稿人可能质疑点。

## 注意事项

- PDF 路径输入应先调用 `research_analyze_paper_pdf`，不要直接把 PDF 原文全文塞入上下文。
- 当前 MVP 只做普通论文 PDF 的文本抽取、章节启发式识别和抽取式分块摘要，不保证公式、图片和表格完整解析。
- 如果信息不足，列出缺失信息，不要伪造作者、年份、venue 或 DOI。
- 生成最终 Markdown 后使用 `research_write_report` 保存到 `notes`。

## 质量检查清单

- 是否说明 PDF 解析限制。
- 是否区分论文原文信息和模型推断。
- 是否覆盖方法、实验、贡献、不足。
- 是否保存最终报告。
